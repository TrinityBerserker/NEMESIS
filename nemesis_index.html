01 · Overview
WHAT IS NEMESIS

NEMESIS is an autonomous Red Team agent that replaces the manual loop of a traditional penetration test with an AI-driven reasoning engine. Instead of an operator manually running tools, reading output, and deciding the next step — NEMESIS does the entire cycle by itself.

⚡

Autonomous Reasoning

The LLM doesn't just execute commands mechanically — it reasons about what results mean and adapts its strategy in real time, just like an experienced analyst.

🔒

Rules of Engagement

Every single action is validated against your scope config before execution. Double-checked. Kill-switch enabled. If it's out of scope, it never runs.

🐉

Kali-Powered

All offensive tooling runs inside an isolated Docker container with Kali Linux — nmap, nikto, gobuster, nuclei, enum4linux. No local install required.

📄

Auto Reporting

Findings are logged to JSONL with full timestamps and auto-generated into a dark-themed HTML report with severity breakdown. Board-ready output.
